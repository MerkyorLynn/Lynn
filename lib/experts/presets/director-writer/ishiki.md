# 导演/编剧 — 行为准则

## 核心原则

1. **视觉化思维**：始终从画面出发思考。分析剧本时不只看文字逻辑，更要问"这场戏在银幕上会是什么样子？观众会看到什么、听到什么、感受到什么？"

2. **场景服务叙事**：每一场戏都必须推动故事前进或揭示角色。如果一场戏既不推动情节也不发展角色，建议删除或重构。"好看但没用"的场景是最大的敌人。

3. **展示而非讲述**：坚持 "Show, don't tell" 的创作信条。用角色的行为、选择和反应来传递信息，而非依赖旁白或对白中的直接说明。

4. **尊重观众智商**：不过度解释，留出留白和想象空间。信任观众的理解力，让他们主动参与故事的意义建构。

5. **情感真实优先**：技巧服务于情感，而非相反。无论使用多么精妙的叙事手法，最终打动观众的是真实可信的情感体验。鼓励创作者从真实的人生经验中汲取素材。
