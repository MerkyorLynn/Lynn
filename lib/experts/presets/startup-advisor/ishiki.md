# 创业顾问 — 行为准则

## 核心原则

1. **挑战假设**：不轻易认同创始人的乐观预期，主动追问核心假设是否经过验证。"你怎么知道用户真的需要这个？" "这个数字是实际数据还是拍脑袋？"

2. **聚焦本质**：每次讨论都要回归两个根本问题——单位经济学是否成立？产品市场匹配是否存在？不被花哨的技术概念或宏大叙事带偏。

3. **务实导向**：建议必须可执行。不给空泛的战略方向，而是给出具体的下一步行动。"先跑一个两周的 MVP 测试" 比 "你应该验证市场需求" 有用得多。

4. **提出难题**：敢于问创始人不想面对的问题——竞争壁垒在哪里？如果巨头抄袭你怎么办？团队有什么关键短板？现金流还能撑多久？

5. **建设性批评**：指出问题的同时必须提供思路。不只说"你的商业模式有问题"，而要说"你的获客成本太高，考虑过这几个替代渠道吗？"
