安装前你可能想知道

我怎么知道我的 Mac 是 Apple Silicon 还是 Intel？

点击左上角苹果菜单，选择“关于本机”。如果看到 M1 / M2 / M3 / M4，就是 Apple Silicon；如果看到 Intel，就是 Intel 版。

Windows 弹出安全提示还能装吗？

可以。当前安装包未做 Windows Authenticode 签名，所以系统可能提示未知发布者。点击“更多信息”后仍可继续安装。

我想看更新说明或校验原始发布记录怎么办？

你可以直接打开 GitHub Release 页面查看所有历史版本、更新说明和备用下载入口。
