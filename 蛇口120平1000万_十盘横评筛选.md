# 深圳蛇口 120㎡ / 1000万改善盘 · 十一盘深挖（密度+品质+位置）

> **权重调整**：密度（容积率/建筑形态）35% + 靠近南山核心 25% + 楼盘品质 25% + 其他 15%
> 配套和学区权重降低，聚焦「住得舒服」「房子本身好」「位置不偏」
> 数据来源：房天下、安居客、贝壳、百度百科、深圳房地产信息网（2026 年 3 月）

---

## 一、核心发现：容积率是分水岭

搜完数据才发现，蛇口 1000 万预算段的十个盘，按容积率可以分成三个完全不同的世界：

| 密度等级 | 容积率范围 | 代表楼盘 | 居住感受 |
|---------|-----------|---------|---------|
| 🟢 低密 | < 2.5 | 山语海（1.87）、兰溪谷二期（<2.0）、鸣溪谷（2.89） | 小高层+大花园，楼间距宽，采光通风好 |
| 🟡 中密 | 2.5-4.0 | 招商雍景湾（2.66）、半山臻境（3.75） | 正常高层社区，密度可接受 |
| 🔴 超高密 | > 7.0 | 万科蛇口公馆（7.3）、水湾1979（7.5）、晗山悦海（7.39）、开云府（7.0） | 综合体/超高层，人口密度极大，等电梯久 |

**这个发现彻底改变了之前的推荐排序。** 上版报告推荐的「综合最优」晗山悦海和「性价比之王」万科蛇口公馆，容积率竟然都超过 7.0，属于超高密度的综合体项目。

## 三、逐盘深挖

### 1. 山语海 — 蛇口密度之王

| 维度 | 数据 |
|------|------|
| 容积率 | **1.87**（蛇口最低，全南山前三） |
| 建筑形态 | 11 栋板楼小高层，最高约 30 层 |
| 总户数 | 606 户（11 栋，平均每栋 55 户） |
| 梯户比 | 2 梯 4 户 |
| 绿化率 | 30-40%（不同来源有差异） |
| 车位 | 727 个，约 1:1.2 |
| 开发商 | 赤湾地产（招商系） |
| 物业 | 赤湾物业，4.7 元/㎡/月 |
| 建筑品质 | 2016 年建成，外立面维护尚可；板楼设计，南北通透 |

**密度优势**：1.87 的容积率意味着楼间距大、采光充足、每层只有 4 户。对比容积率 7.0+ 的综合体，山语海的居住体验完全不同。小区花园面积大，背靠小南山，部分高层山海双景。

**品质短板**：开发商是赤湾地产（非一线品牌），物业是赤湾物业（非招商物业），公共区域维护水平一般。物业费 4.7 元属于中等。

**位置**：赤湾片区，距后海/科技园核心区约 8km，通勤需 30-40 分钟。这是最大的劣势。

**120㎡4 房预估总价**：850-1000 万（单价 8.3 万）

### 2. 兰溪谷二期 — 低密+品牌+山景

| 维度 | 数据 |
|------|------|
| 容积率 | **< 2.0**（蛇口最低之一） |
| 建筑形态 | 11 栋小高层 + 叠拼别墅 |
| 总户数 | 537 户（二期），含别墅产品 |
| 梯户比 | 1 梯 2 户 / 2 梯 2 户 |
| 绿化率 | **50%**（蛇口最高） |
| 车位 | 735 个，约 1:1.4 |
| 开发商 | **招商蛇口** |
| 物业 | **招商物业**，4.5-5 元/㎡/月 |
| 建筑品质 | 2007 年建成，招商系品质，外立面和园林维护良好 |

**密度+品质双优**：容积率 <2.0 + 绿化率 50% + 招商品牌 + 招商物业 + 1梯2户。这几个指标组合在一起，在蛇口 1000 万预算段没有对手。兰溪谷二期是蛇口半山区的高端低密社区，西靠大南山，整体气质更接近别墅区的质感。

**位置**：蛇口半山区（沿山路），比赤湾更靠近蛇口核心。距后海约 5km。

**关键问题**：120㎡3 房挂牌价约 950-1100 万，可能略超预算。如果看 89㎡3 房，约 700-800 万，性价比极高。另外楼龄 19 年需要实地查看维护状况。

### 3. 鸣溪谷 — 蛇口半山隐藏的低密小盘

| 维度 | 数据 |
|------|------|
| 容积率 | **2.89** |
| 建筑形态 | 6 栋小高层，**最高仅 16 层** |
| 总户数 | **216 户**（蛇口改善盘中最少，6 栋平均每栋仅 36 户） |
| 梯户比 | **1 梯 2 户** |
| 绿化率 | **70%**（蛇口最高，远超所有竞品） |
| 车位 | 235 个，约 1:1.08 |
| 开发商 | 深圳市崇诚实业有限公司（非一线品牌） |
| 物业 | 保利物业，3.0 元/㎡/月（蛇口最低物业费之一） |
| 建筑品质 | 1996-2006 年分批建成，依山就势，南低北高 |
| 参考均价 | 102,164-118,068 元/㎡（各平台差异大，取中约 10.5 万/㎡） |
| 在售 | 7-12 套（流通量极低） |
| 学位 | 育才体系（育才一小/三小、育才一中） |

**密度+户数双优**：2.89 的容积率本身不算极低，但关键是 **6 栋楼总共只有 216 户，最高 16 层，1 梯 2 户**。这个户数密度在全蛇口改善盘中是最低的。对比招商雍景湾 1499 户/7 栋，鸣溪谷的社区体量只有它的七分之一。这意味着几乎不用等电梯、花园基本不拥挤、邻居少社区安静。

**绿化率 70% 是碾压级数据**：蛇口绿化率排名：鸣溪谷 70% > 兰溪谷二期 50% > 半山臻境 40% > 山语海 30-40%。70% 的绿化率在蛇口基本找不到第二个。加上三面环山（大南山脚下，工业六路与沿山路交汇处），整个小区长在山里的感觉。

**位置优势**：蛇口半山区，和兰溪谷二期同一片区。沿山路/工业六路，距后海约 5km。**比赤湾（山语海、半山臻境）近 3km**，通勤时间少 10-15 分钟。距离蛇口核心区（海上世界）步行约 15 分钟。

**品质短板**：开发商崇诚实业不知名，物业是保利物业（非招商系），公共区域维护水平需要实地验证。楼龄最早一批 1996 年（30 年），最晚 2006 年（20 年），整体偏老。物业费 3 元是蛇口最低之一，但也可能意味着物业服务水平有限。

**户型与预算匹配**：

| 户型 | 面积 | 预估总价 |
|------|------|---------|
| 2 房 | ~55㎡ | ~580-650 万 |
| 3 房 | ~86㎡ | ~780-900 万 |
| 3+1 房/4 房 | ~120-130㎡ | ~1050-1200 万 |
| 顶楼复式 | ~185㎡ | ~1500 万+ |

⚠️ **关键问题**：86㎡3 房约 780-900 万，预算内可入但面积偏小。120-130㎡大户型约 1050-1200 万，**超出 1000 万预算**。鸣溪谷的大户型单价偏高，流通量又极低（仅 7-12 套在售），可能需要长时间等笋盘。

**与兰溪谷二期对比**：

| 维度 | 鸣溪谷 | 兰溪谷二期 |
|------|--------|-----------|
| 容积率 | 2.89 | <2.0 |
| 绿化率 | **70%** | 50% |
| 总户数 | **216 户** | 537 户 |
| 梯户比 | 1 梯 2 户 | 1 梯 2 户 / 2 梯 2 户 |
| 开发商 | 崇诚实业 | **招商蛇口** |
| 物业 | 保利 3 元 | **招商物业 4.5 元** |
| 楼龄 | 1996-2006（最老 30 年） | 2007（19 年） |
| 学区 | 育才 | 育才 |
| 位置 | 蛇口半山同一区域 | 蛇口半山同一区域 |
| 86㎡3 房总价 | 780-900 万 | ~700-800 万 |
| 120㎡大户型 | 1050-1200 万 | 950-1100 万 |

**结论**：鸣溪谷是一个「密度极低、体量极小、绿化极高、位置在蛇口半山」的隐藏小盘。86㎡3 房在预算内但面积偏小，120㎡大户型超预算。如果接受 86㎡，鸣溪谷的居住密度和安静程度甚至优于兰溪谷二期（户数只有它的 40%）。但品牌和楼龄是短板。**适合追求极致安静低密、不在乎品牌、预算 800-900 万的买家。**

### 4. 招商雍景湾 — 中密+蛇口核心+品牌天花板

| 维度 | 数据 |
|------|------|
| 容积率 | **2.66** |

| 梯户比 | 2 梯 4 户 |
| 绿化率 | 33% |
| 车位 | 1666 个，约 1:1.1 |

**密度中等偏上**：2.66 的容积率属于正常高层社区密度，不低但也不算高。问题在于 1499 户/7 栋 = 平均每栋 214 户，高峰期电梯压力大。

**购买建议**：一期 105㎡3 房砍到 900 万以内可入。二期 116㎡4 房挂牌约 1200 万，砍到 1000 万需要等笋盘。

### 6. 万科蛇口公馆 — 超高密综合体，密度硬伤

| 维度 | 数据 |
|------|------|
| 容积率 | **7.3**（超高密） |
| 总户数 | **1934 户**（含回迁房） |
| 绿化率 | **12%**（蛇口最低，几乎没有花园） |
| 交易权属 | 商品房 **/ 动迁安置房 / 使用权** |

**重大问题暴露**：7.3 的容积率 + 绿化率仅 12% + 含动迁安置房 + 1934 户挤在 5 栋楼里。这不是普通住宅社区，是超高层综合体，其中包含回迁房。上版报告将其列为「性价比之王」是错误的。便宜的代价是极差的居住密度。

**位置确实是优势**：东角头站 110m，蛇口湾厦片区，距后海 4km。但位置好弥补不了密度差的硬伤。

**结论：排除出推荐列表。** 除非你是纯投资出租，否则不建议自住。

### 7. 晗山悦海 — 超高密，但住宅部分极稀缺

| 维度 | 数据 |
|------|------|
| 容积率 | **7.39**（超高密综合体） |
| 总户数 | 住宅仅 **96 套**（C 栋住宅，30 层） |
| 梯户比 | **2 梯 2 户**（住宅部分） |
| 占地面积 | 1.93 万㎡ |
| 建筑面积 | **23.1 万㎡**（综合体含公寓/办公/商业） |
| 物业费 | 7.5 元/㎡/月 |

**关键区分**：晗山悦海整体容积率 7.39 是因为它是综合体（住宅+公寓+办公+商业）。但住宅部分只有 C 栋 30 层共 96 套，2 梯 2 户。住宅的梯户比和户数配比其实很好，但整栋楼被综合体包围，花园空间极小（占地 1.93 万㎡建了 23 万㎡建筑面积）。

**结论**：如果只看住宅部分的 2 梯 2 户，体验可能还行。但 7.39 的整体容积率意味着下楼就是商业和办公人流，没有安静的社区花园氛围。追求低密居住体验的话，这个盘不符合要求。

### 10. 金众云山海 — 中密但品牌弱

| 维度 | 数据 |
|------|------|
| 容积率 | 西区 **3.45** / 东区 **4.86** |
| 总户数 | 西区 330 / 东区 480 |
| 梯户比 | 西区 2 梯 4 户 / 东区 **3 梯 6 户** |
| 开发商 | 金众地产（非一线品牌） |
| 物业费 | 7.8 元（偏高） |

**密度可接受（西区）**：3.45 的容积率在蛇口属于中等，2 梯 4 户也够用。但东区 4.86 + 3 梯 6 户就明显偏挤了。

## 六、核心结论

按你给的权重（密度>位置>品质，配套学区降低），排名逻辑很清晰：

1. **先排除所有容积率 >5.0 的盘**（万科蛇口公馆、晗山悦海、水湾1979、开云府），这 4 个盘属于超高密综合体，居住体验跟低密住宅完全不同
2. **剩下的 7 个盘按位置排序**：招商雍景湾（蛇口核心）> 兰溪谷/鸣溪谷（蛇口半山）> 半山臻境/山语海/金众云山海（赤湾）
3. **同位置按品质排序**：招商品牌 > 金众品牌；全新盘 > 老盘

**最终推荐**：如果必须卡 1000 万且要 120㎡，**招商雍景湾 116㎡4 房**（等笋盘砍到 1000 万）或**山语海 120㎡4 房**（850-950 万，密度最低）。如果面积可以降到 89-105㎡，**兰溪谷二期 89㎡3 房**（~750 万）是品质+密度+位置的最优解。
